Здравствуйте, представляю вашему вниманию химический калькулятор молекулярной массы вещества.
Реализована программа в виде приложения с графическим интерфейсом. 

В программе имеется 4 поля ввода. 
Два поля для ввода атомов и два для ввода индексов атомов.

Пример ввода:
Ввод формулы: формула вводится строго с большой буквы без пробелов.
Пример: H2O, Na2O, FeO3.
Расчет производится по нажатию на "to count".


Данная программа отличается от предыдущей тем что тут реализован однострочный ввод формулы, 
но теперь нет нужды ставить пробелы в формуле и можно вводить формулы любой длины из-за этого
убрано вычисление мосовой доли весчества.
